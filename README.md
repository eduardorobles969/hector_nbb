## 🧩 Features principales

- 🏋️ **Dashboard de clientes** – Seguimiento de progreso en tiempo real.  
- 🍽️ **Planes de dieta personalizados** – Adaptados a calorías, macros y objetivos.  
- 📝 **Rutinas inteligentes** – Mesociclos, progresiones y entrenamientos híbridos.  
- 🔔 **Notificaciones motivacionales** – Recordatorios y frases épicas para mantenerte on fire.  
- ☁️ **Integración con Firebase** – Autenticación, Firestore, Storage y Realtime Database.
